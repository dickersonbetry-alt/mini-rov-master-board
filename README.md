# Mini-ROV 主控板程序开发
E唯创新团队第四周考核A题

## 项目简介
本项目是Mini-ROV水下机器人的主控板控制程序，基于STM32F103C8T6开发，实现了串口通信、四电机推力分配、PWM输出和电流计数据采集功能。

## 已实现功能
1.  通信协议：支持下行请求帧响应和上行数据帧发送，严格遵循手册定义的帧格式和异或校验
2.  电机控制：完成推力-PWM曲线拟合，实现三自由度矢量推力分配
3.  电流采集：通过I2C读取INA226电流数据并回传

## 使用指南
1.  用keil打开工程
2.  连接ST-Link下载程序
3.  串口调试助手设置115200波特率，发送`FA AF 52 FB BF`测试通信
